# House Price Prediction 🏠

A Machine Learning project that predicts house prices based on 
property features using Linear Regression and Random Forest models.

## 📦 Dataset
- Source: Kaggle — Housing Prices Dataset
- Records: 545 houses
- Features: 13 columns (area, bedrooms, bathrooms, etc.)

## 🛠️ Tools & Libraries Used
- Python 3
- Google Colab
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## ✅ Tasks Completed
- Data Loading & Exploration
- Data Cleaning & Preprocessing
- Model Building (Linear Regression & Random Forest)
- Data Visualization (3 Charts)
- Insights & Summary

## 📊 Model Performance
- Random Forest outperformed Linear Regression
- R² Score: 0.60+ (explains 60% of price variation)

## 🔍 Key Findings
- Area is the strongest predictor of house price
- Bathrooms and stories also strongly influence price
- Air conditioning adds more value than number of bedrooms
